======= Site Comercial =======

Linguagem do código:
Utilizei uma estrutura híbrida no HTML. Já no CSS, optei por usar nomes em português nas variáveis do :root para ter maior agilidade no desenvolvimento.


Parametros dos banners
deskto => 1920x960px => no PS salvar para web == JPEG, quality 60/65,progresivo marcado e converter p/ sRGB-cor do monitor
mobile => 1200x600px => no PS salvar para web == JPEG, quality 60/65,progresivo marcado e converter p/ sRGB-cor do monitor
TinyPNG.com => aite para diminuir o tamanho da imagem ou banner
